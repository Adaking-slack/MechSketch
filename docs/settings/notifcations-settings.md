# Notifications Settings

## Overview
Controls user notification preferences.

## UI Elements
- Toggle: Email Notifications
- Toggle: Push Notifications
- Toggle: Product Updates
- Button: Save Preferences

## Behavior
- Toggles reflect current saved settings
- Changes are temporary until saved

## Actions
- On Save Preferences:
  - Persist settings
  - Show success: “Preferences updated”

## Notes
- Clear ON/OFF states must be visually obvious
