# Appearance Settings

## Overview
Controls visual theme and styling preferences for the app.

## UI Elements
- Toggle: Light Mode
- Toggle: Dark Mode
- (Optional) Toggle: System Default

## Behavior
- Only one mode can be active at a time
- Theme changes should apply instantly across the app

## Actions
- On change:
  - Update global theme state
  - Persist user preference

## Notes
- Must sync across all pages
