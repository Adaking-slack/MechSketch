# Layout Settings

## Overview
Defines global layout behavior across the application.

## UI Rules

### Page Structure
- All pages must be centered horizontally
- Max content width must be consistent across screens
- Avoid random shifts in alignment

### Spacing System
- Use consistent spacing scale (8px base unit)
- Keep vertical rhythm consistent across pages

### Components
- No random layout variations per page
- Buttons, inputs, and cards must follow shared spacing rules

## Behavior
- Layout must remain consistent across navigation
- No page should override global alignment rules unless explicitly defined

## Notes
- This file ensures UI consistency across the entire app
